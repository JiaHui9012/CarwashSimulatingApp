# CarwashSimulatingApp

IDE used: Android Studio

This application simulates a simple car wash service workflow where users can register their vehicles and retrieve them after service using a unique code.

### Features
- Users can enter a car plate number to register for the car wash
- System generates a unique code for each car
- Cars are added into a queue for processing
- After the wash is completed, users can retrieve their car using the assigned code
- A queue list can be viewed by users

### Purpose
This project demonstrates basic queue management, unique identifier generation, and workflow handling in a real-world scenario.

## How To Run:

Download and unzip the code

Import this folder CarwashSimulatingApp into Android Studio

Click on the run button to run the app
